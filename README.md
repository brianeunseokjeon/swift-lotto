2. 로또
- 로또 하나의 가격은 1000원입니다
- 로또 번호는 1~45 사이 숫자입니다.
- 로또 번호는 크기가 작은 순, 중복 불가 6개 생성됩니다.
- 동일한 로또는 가능합니다
- 3개 일치 5000원, 4개 일치 50000원, 5개 일치 150만원, 6개일치 20억입니다.
- 수익률은 총 당첨금액 / 총 사용금액 입니다
